구현한 내용 :
• 1: 애니메이션 1
• 2: 애니메이션 2
• 3: 애니메이션 3
• t: 조명을 켠다/끈다.
• c: 조명 색을 바꾼다. 최소 3개의 다른 색을 적용해본다.
• y/Y: 카메라가 바닥의 y축을 기준으로 양/음 방향으로 회전한다. (공전, 다시 누르면 회전을 중지한다.) ==> 마우스와 WASD 이동으로 대체
• +/-: 육면체 이동하는 속도 증가/감소
• r: 모든 값 초기화 (새롭게 가로세로 값을 입력받아 애니메이션 시작한다)
• q: 프로그램 종료
